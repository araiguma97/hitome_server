# hitome_server

## Usage

1. Add "config.php" to directory html/
    ```php
    // config.php
    <?php
        $dsn = "mysql:host=localhost;dbname=dbname;charset=utf8";
        $user = 'user';
        $password = 'password';
    ?>
    ```

1. Add "config.py" to directory script/
    ```python
    # config.py
    user='user'
    password='password'
    host='host'
    database='database'
    ```
